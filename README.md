# 🚀 Streamlit Learning Journey

This repository documents my **journey of learning Streamlit** by building interactive **data applications, dashboards, and web interfaces using Python**.

The goal of this project is to practice and understand how to create **interactive data apps with Streamlit**, work with datasets using **Pandas**, and build simple dashboards.

All applications in this repository are built using **Python and Streamlit**, with each chapter focusing on a specific concept.

---

# 📚 What I'm Learning

Through this project I am learning how to:

* Build interactive web apps using **Streamlit**
* Create UI components like buttons, sliders, dropdowns, and forms
* Design layouts using columns, sidebars, and expanders
* Work with datasets using **Pandas**
* Build simple **data dashboards**
* Deploy Streamlit applications to the web

---

# 🧰 Technologies Used

* Python
* Streamlit
* Pandas

# ▶️ How to Run the Project

### 1️⃣ Install dependencies

```bash
pip install streamlit pandas numpy
```

### 2️⃣ Run a Streamlit file

```bash
streamlit run chapter-one.py
```

### 3️⃣ Open in browser

```
http://localhost:8501
```
